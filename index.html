Resumo

Sou apaixonado pelo mundo do desenvolvimento. Descobri a programação em 2018 com a linguagem Python, aprendendo o básico de algoritmos e me aventurando por este novo mundo. A felicidade ao ver o algoritmo rodando é o que me mantém motivado a resolver os problemas. Em 2019 fiz uma escolha muito importante: Entrei para o IFSUL no curso técnico de informática, onde iniciei para valer nesse universo. No mesmo ano, participei da Hackathon dos IFs, onde aprendi a criar soluções rápidas para problemas complexos, o que veio bem a calhar na área do desenvolvimento. Em 2020 estudei Java, C, HTML, CSS e Javascript, dando passos mais profundos. Aproveitando a pandemia, utilizei o ano de 2021 para mergulhar de cabeça no desenvolvimento, estudando Wordpress, SQL, PHP, Flutter, git e Github e fazendo meu primeiro CRUD. Além disso, participei de uma bolsa de estudos onde desenvolvi um pequeno sistema de realidade aumentada para o site de uma MEI local, o que viria a me trazer também o segundo lugar da MOCITEC na categoría Informática pelo trabalho realizado. Hoje busco minha primeira oportunidade profissional, desejo me tornar um desenvolvedor Fullstack e impactar vidas através da programação.
